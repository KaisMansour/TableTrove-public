# TableTrove
Plateforme web et mobile de réservation de tables dans des restaurants, développée dans le cadre d’un projet de session.

## Description
TableTrove est une application inspirée d’OpenTable permettant aux utilisateurs de consulter des restaurants et de réserver des tables selon les disponibilités.
Le projet inclut la conception d’une API REST, la gestion d’une base de données SQL et le déploiement de l’application.

## Technologies
- JavaScript
- HTML / CSS
- SQL
- API REST
- Application mobile

## Fonctionnalités
- Réservation de tables
- Gestion des restaurants
- Gestion des utilisateurs
- Backend API REST
- Déploiement de l’application
